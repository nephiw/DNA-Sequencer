Detailed Description

The FileScanner class is used to scan a text file and retrieve tokens based on a specific delimiter or retrieved specific forms of data delimited by the provided delimiter.

void Utilities::FileScanner::undoScan ( ) [virtual]

When retrieving data from a file the current position in the file is maintained. The history of each modification of the current position in the file is saved, allowing the user to undo any previous scanning events. For example if the token "Hello123" was read using getNextToken(), calling undoScan() would result in the next call to getTextToken() resolving to the same "Hello123" token.
