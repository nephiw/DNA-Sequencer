Detailed Description

Takes a list if SequenceVO objects, and constructs a matrix of their scored pairwise comparisons.

This relies on the pyd as well. It gets its settings from the Model. The pairwise comparisons occur in multiple processes, but is still blocking system wide until they are complete. It queries the system for the number of CPU's, and uses this to determine the number of processes to run.
