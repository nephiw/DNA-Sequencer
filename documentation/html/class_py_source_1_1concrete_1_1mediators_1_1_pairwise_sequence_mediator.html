Detailed Description

PairwiseInterface.

Once initialized, PairwiseSequenceMediator configures the combo boxes and sets up virtual event listeners on the buttons. When the application requests a state save, PairwiseSequenceMediator creates a PairwiseSelectorVO to store the sequencer's settings.
