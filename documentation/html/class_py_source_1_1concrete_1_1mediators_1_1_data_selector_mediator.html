Detailed Description

Mediates DataSelectorInterface.

Once initialized, DataSelectorMediator sets up virtual event listeners on the buttons. When the application requests a state save, PairwiseSequenceMediator creates a DataSelectorVO to store the data objects that will be sequenced.
