Detailed Description

Mediates an instance of tkinter's show info box.

It is essentially an alert box. When non-view elements want to display a message to the screen, a notification will be sent that will result in this mediator creating and displaying the message sent to it. For more information in tkinter, visit this site: http://www.tkdocs.com/
